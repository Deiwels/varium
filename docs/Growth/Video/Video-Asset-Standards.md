# Video Asset Standards

> Part of [[Home]] > Growth | See also: [[Growth/Video/README|Video]], [[Growth/Creative/Approved-Claims-and-Angles|Approved Claims and Angles]], [[08-Runbooks/Growth/Creative-Production-Workflow|Creative Production Workflow]]

## Purpose

Set the minimum rules for scripts, demo representations, and promo videos so video work stays aligned with real product truth.

## Standards

- every script must link back to approved product or growth docs
- feature claims must match current product truth
- risky claims route through AI 7 before use
- brand-sensitive final assets still require Owner approval

## Do Not

- imply unsupported automation
- present roadmap items as live features
- use compliance-sensitive claims without a documented source
