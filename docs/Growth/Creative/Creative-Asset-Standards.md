# Creative Asset Standards

> Part of [[Home]] > Growth | See also: [[Growth/Creative/README|Creative]], [[Growth/Creative/Approved-Claims-and-Angles|Approved Claims and Angles]], [[08-Runbooks/Growth/Creative-Production-Workflow|Creative Production Workflow]]

## Purpose

Set the minimum rules for static marketing visuals so the team can create ads quickly without drifting away from real product truth.

## Standards

- every creative must map to an approved angle or source-of-truth note
- visuals must not imply features that do not exist
- landing-page creative must not conflict with product UI ownership
- brand-sensitive final assets still require Owner approval

## Required Metadata

- target audience
- campaign angle
- CTA intent
- source-of-truth links
- claim-risk note if applicable
