## Hard rule

- If AI 4 posts a review in GitHub first, that review is **not team-visible enough** until it is mirrored here.
- A 4-AI Plan Review Gate item for AI 4 stays blocked until the relevant mirror entry exists here and is linked from [[Tasks/In Progress|In Progress]].
- The mirror should be created in the same session in which the GitHub review is noticed.
