# System Changes Log

> Part of [[Home]] > Decisions | See also: [[10-Decisions/Decisions-Index|Decisions Index]], [[00-System/Self-Improvement-Layer|Self-Improvement Layer]], [[11-Reference/Templates/Improvement-Proposal-Template|Improvement Proposal Template]]

## Purpose

Canonical location for approved system-level improvements and operating-model changes.

Use this lane when a change affects:

- workflows
- templates
- AI roles
- system rules
- KPI measurement
- escalation behavior
- self-improvement process

## Required Logging Rule

Every approved system improvement should leave behind a decision note here that records:

- what changed
- why it changed
- when it changed
- who approved it

## Suggested Flow

1. detect problem through KPI, incidents, confusion, or repeated friction
2. draft proposal using [[11-Reference/Templates/Improvement-Proposal-Template|Improvement Proposal Template]]
3. review through the required lane reviewers
4. apply the approved change
5. log the final change here

## Owner Lane

- **AI 3** for structure and review coordination
- **Owner** for major/risky approval
- relevant owning lane for the actual content of the change
