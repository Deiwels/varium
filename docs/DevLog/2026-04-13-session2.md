## New Features

### Site Analytics Widget
- Tracks every visit to booking page
- Auto-detects source: Instagram, Google, Facebook, TikTok, Twitter, Direct
- Supports `utm_source` URL parameter
- Dashboard widget: total visits, source bars, daily chart
- Backend: `POST /public/analytics/:wsId`, `GET /api/analytics/summary`

### Email System (Resend)
- **Resend API** integration for email sending
- `sendEmail()` + `vuriumEmailTemplate()` with 6 themed templates
- **Booking confirmation email** with Reschedule/Cancel buttons
- **Password reset** via email (forgot password flow)
- **Reset password page** (`/reset-password`) in Vurium dark cosmos style
- All emails match selected site template (Modern, Classic, Bold, Dark Luxury, Colorful, Custom)
- Business logo in emails (uploaded in Settings)
- DNS records added to Namecheap for vurium.com (DKIM, SPF, DMARC)

### Password Reset Flow
- `POST /auth/forgot-password` — generates token, sends email
- `POST /auth/reset-password` — validates token, updates password
- Token expires in 1 hour
- "Forgot password?" on signin page
- Styled reset page with password strength indicators

### Manage Booking (Client Self-Service)
- `/manage-booking?token=...` page
- Clients can **reschedule** (pick new date/time)
- Clients can **cancel** booking
- Linked from email confirmation buttons
- `client_token` generated for each booking

### Delete Account
- Backend: owner deletes entire workspace + all subcollections
- Non-owner: soft-delete
- UI in Settings → Accounts

### Duplicate Prevention
- **Signup**: checks all workspaces for existing email → 409 with "Sign In" link
- **Clients**: dedup by phone_norm AND email
- **Public booking**: links to existing client by phone/email

### Plan Enforcement
- Individual plan: `member_limit: 1` enforced on backend
- Frontend: "Upgrade to add team" link instead of "+ Add member"
- Custom plan added to signup ($99)

### Hero Image Upload
- File upload in Site Builder (replaces URL input)
- Compressed to max 1200px JPEG
- Preview with remove button

### Business Logo Upload
- Settings → General → Logo
- Max 5MB, compressed to 256px PNG
- Used in emails and booking page

## Bug Fixes

### Critical
- **React #418 hydration** — deferred all localStorage reads to useEffect (Shell, Dashboard)
- **Shell status hydration** — init as 'loading', set optimistic in useEffect
- **Booking modal crash** — `await onSave()` so button stays in "Saving" state
