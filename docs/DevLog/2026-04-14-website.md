## Backend Changes

### `POST /contact` Endpoint
- Public endpoint (no auth required)
- Rate limiting: 1 request per minute per IP
- Input validation: name, email, message required; email format check
- HTML sanitization via `sanitizeHtml()`
- Sends formatted email to `support@vurium.com` via Resend
- Uses `vuriumEmailTemplate()` with modern theme
