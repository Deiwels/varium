## Purpose

Provide the canonical template library for structured work across the VuriumBook AI Operating System.

## Rule

No AI should create a free-form note when a matching template already exists here.

Short version:

- no template = no work
- if a template exists, use it
- AI 3 may reject or send back notes that ignore an available template

## Enforcement

- AI creating a structured note must start from the matching template when one exists.
- AI 3 is the template-discipline enforcer for task/process notes and may block chaotic notes from being treated as canonical.
- If a template is missing, create or approve the template before normalizing ad hoc notes.
